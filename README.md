# Intelligent-DDoS-Attack-Detection-System
🚀 Intelligent DDoS Attack Detection System

📌 Project Overview

This project focuses on detecting Distributed Denial-of-Service (DDoS) attacks, including unknown (zero-day) attacks, using advanced machine learning techniques like:

- Open Set Recognition (OSR)
- Reciprocal Points Learning (RPL)

Traditional systems fail to detect unknown attacks, but this system overcomes that limitation.

---

🎯 Objective

- Detect both known and unknown DDoS attacks
- Classify traffic into:
  - Normal Traffic
  - Known Attack
  - Unknown Attack
- Improve network security using intelligent models

---

🧠 Key Concepts

- Open Set Recognition (OSR)
- Reciprocal Points Learning (RPL)
- Deep Learning (CNN Models)
- Network Traffic Analysis

---

⚙️ Technologies Used

🏗️ System Architecture

The system works in the following steps:

1. Data Collection (Network Traffic)
2. Data Preprocessing
3. Feature Extraction
4. Machine Learning Model Training
5. Open Set Recognition
6. Reciprocal Points Learning
7. Prediction & Detection
8. Alert Generation

👉 As shown in your report (Page 16), the architecture includes preprocessing → feature extraction → classification → alerts.
